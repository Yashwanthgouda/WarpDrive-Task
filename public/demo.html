How to Use the Platform

Admin Dashboard

Access the admin interface to manage events, students, and view reports.

1. Create Events

Add new events with details like type, date, location, and capacity.

2. Manage Students

View and manage student profiles across different colleges.

3. View Reports

Generate comprehensive analytics and reports.

Open Admin Dashboard

Student Portal

Students can browse events, register, and provide feedback.

1. Register

New students can create an account with their college details.

2. Login

Existing students can login using their college and student ID.

3. Register for Events

Browse and register for available events in their college.

4. Provide Feedback

Submit feedback and ratings for attended events.
